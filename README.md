# OpenCV Learning 🚀

This repository documents my journey of learning **Computer Vision** using **Python** and **OpenCV**. Each lesson contains small, focused programs that help me understand the fundamentals before moving on to advanced topics like face detection, hand tracking, and object detection.

## 🛠️ Technologies Used

- Python 3
- OpenCV (cv2)
- Visual Studio Code
- Git & GitHub

## 📚 Topics Covered

### Image Processing
- Reading and displaying images
- Image properties (shape, size, channels)
- Color space conversion (BGR, RGB, HSV)
- Grayscale images

### Video Processing
- Reading videos
- Webcam capture
- Recording videos

### Image Manipulation
- Drawing lines, rectangles, circles
- Writing text on images
- Image resizing
- Image cropping

### Color Detection
- HSV color space
- Color masking
- Bitwise operations

### Image Processing Techniques
- Gaussian Blur
- Edge Detection (Canny)
- Thresholding
- Contour Detection
- Bounding Rectangles

### Face Detection
- Haar Cascade Classifier
- Detecting faces in images
- Drawing bounding boxes around detected faces

### Hand detection
- Mediapipe
- Detecting hands by drawing skeletons
- Counting fingers

###Hand gesture
- Detecting the volume range of the system in dB
- Controlling the volume by mapping it to Pycaw
- Air canvas random drawing with erasing and color changing options
- Mouse detection and clicking

###YOLO AI-integration
- Detecting objects by using YOLO
- Person detection and moniotring confidence level of YOLO
- Exam hall monitoring by phone detection
